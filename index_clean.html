🎨 Theme Demo

Δοκιμαστική σελίδα για το Apple-inspired theme system με δυνατότητα εναλλαγής μεταξύ φωτεινού και σκούρου mode.

Χαρακτηριστικά του Theme System

  • 🌙 Dual Mode: Φωτεινό και σκούρο theme
  • 🎯 Apple-inspired: Αισθητική εμπνευσμένη από τις συσκευές Apple
  • ⚡ Auto-switch: Αυτόματη εναλλαγή με το κουμπί toggle
  • 💾 Memory: Αποθήκευση προτίμησης στο localStorage
  • ⌨️ Shortcuts: Keyboard shortcut (Ctrl/Cmd+Shift+D)
  • ♿ Accessibility: Υποστήριξη για screen readers

Οδηγίες Χρήσης

Για να εναλλάξετε μεταξύ φωτεινού και σκούρου mode:

  1. Κάντε κλικ στο κουμπί 🌙/☀️ στην πάνω δεξιά γωνία
  2. Ή χρησιμοποιήστε το keyboard shortcut: Ctrl+Shift+D (ή Cmd+Shift+D σε Mac)

Η προτίμησή σας θα αποθηκευτεί αυτόματα και θα εφαρμοστεί σε όλες τις σελίδες του συστήματος.
