üîß JavaScript API

The theme switcher exposes a simple API for programmatic control:

// Get available themes
const themes = window.themeSwitcher.getAvailableThemes();

// Get current theme
const current = window.themeSwitcher.getCurrentTheme();

// Set specific theme
window.themeSwitcher.setTheme('apple');

// Listen for theme changes
document.addEventListener('themeChanged', (e) => {
    console.log('Theme changed to:', e.detail.themeName);
});
