Δοκιμή Theme Toggle

Αυτή είναι μια δοκιμαστική σελίδα για το Apple-inspired dual mode theming. Μπορείς να εναλλάσσεις μεταξύ φωτεινού και σκούρου mode χρησιμοποιώντας:

  • Το κουμπί toggle στην πάνω δεξιά γωνία (🌙/☀️)
  • Το πληκτρολογικό shortcut: Ctrl+Shift+D (ή Cmd+Shift+D σε Mac)
  • Το script αποθηκεύει την προτίμηση σου στο localStorage

Χαρακτηριστικά του Theme System

Το σύστημα themes έχει τα εξής χαρακτηριστικά:

  1. Αυτόματη ανίχνευση: Ανιχνεύει την προτίμηση του συστήματος (light/dark)
  2. Αποθήκευση προτίμησης: Θυμάται την επιλογή σου για επόμενες επισκέψεις
  3. Smooth transitions: Μαλακές μεταβάσεις μεταξύ των modes
  4. Accessibility: Υποστήριξη για screen readers
  5. Keyboard shortcuts: Πληκτρολογικά shortcuts για γρήγορη εναλλαγή

Code Example

// Παράδειγμα χρήσης του theme toggle
const toggleButton = document.querySelector('.mode-toggle');
toggleButton.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
});
